7.1. 学生证

不必要等到人到加拿大之后才申请，可以提前办理，上传符合规定的照片，几个工作日后，就可以到学校的指定地点去领域。

学生证里可以充值，用于购买食物，打印，洗衣等等。学生证也是UBC的门禁卡，凭借学生证可以进入会议室，去图书馆借书，购买自动售货机的商品等等。

Note

Official Website Link: Apply for UBC student card

7.2. 学生邮箱

这是和学生证同样重要的事。UBC的校内以及校外有很多与学生相关的福利，需要通过学生邮件进行身份认证，尽量早一点申请。这个也可以现在就申请，不必要等到人到加拿大之后才申请。

Note

Official Website Link: Apply for UBC student email

7.3. SIN

这是加拿大政府用来识别个人身份和追踪其个人保险、税务和就业记录的唯一标识。一个人有且仅有一个。因为我们的课程包含了实习，所以这是一个必须申请的事。注意，这不是一张卡，而是9个数字。虽然办理下来后，工作人员确实会给你一张印有这个号码的纸，但那张纸只是一张普通的A4纸，重要的是这个号码。请千万注意保密，不要发布在网络上，或者告诉除雇主之外的其他人，除非在银行等机构办理相关业务时。

办理的地点是Service Canada Centre，有很多网点，可以在Google上搜一家距离自己最近的。办理所需的材料也是护照、学签大签、证明居住地址的文件，可能还需要提供在读证明。办理SIN接受walk in，但与BCID一样，强烈建议提前预约之后再去。否则现场你可能会等到怀疑是不是时间停止了，别问我是怎么知道的。

Note

Official Website Link: Apply for SIN

7.4. BCID

如果你不打算考取本地的驾照，这是你在读书期间唯一能获得本地政府发布的带照片的ID。拥有了BCID之后，你可以把护照锁起来了。基本上所有境内需要身份认证的地方，你出示这张ID就可以了。

申请地点是ICBC，注意，不是CIBC那个银行。申请过程非常简单，凭借护照、学签大签、证明居住地址的文件，去办理驾照的地方申请，会在10个工作日左右寄到你的邮箱。可以接受walkin，但一般是要求在某个特定的时段，所以最好是提前在网上预约好时间。

Note

Official Website Link: Apply for BCID

7.5. Driver's License

如果你打算买车，可以申请本地的驾照。如果你已经有美国等地方的驾照，可以通过一些paperwork直接更换，但如果你持有的是其他国家的护照，则需要重新考试。BC的驾照分为L，N和5级。如果你此前有驾照，且有一定年限的驾驶经验，可以不用从零开始考试。

注意，如果你以前的驾照语言不是英文，不用先在本国进行翻译，因为温哥华不认可。本地驾照必须经过ICBC授权过的翻译机构进行翻译，费用大概是10-30刀。如果你打算考本地驾照，或通过本国驾照加上学签大签租车，需要进行驾照翻译。
