购物
========
.. contents:: 
   :local:
   :depth: 2

----

超市
------------

温哥华有很多连锁超市，能满足日常生活所需，比如Save on foods, Safeway, Walmart，Costco等等。如果是家具的话，也有宜家、SleepCountry等。


----

大统华
------

大统华YYDS，让你有一秒回国的感觉。超市里应有尽有，从老干妈到辣条，只有你想不到没有他办不到。强烈推荐你找时间去一趟，高低能缓解下思乡之愁。大统华在温哥华有不少店面，Richmond的这家比较大。更好的是，这家步行范围内就有一家ICBC和Canada Service Center，也就是说办BCID和SIN卡可以一起搞定。此外，大统华还提供外卖服务。

----

网购
---------------

加拿大最主流的网购平台依然是Amazon，最快的可以当天达，一般是一到两天。此外，上面提到的不少超市，比如Walmart等也有自己的网购渠道，通过App就能下单。

从价格来讲，最便宜的是今年刚刚在加拿大开始运营的Temu。但缺点是，一是从下单到收到货，大概需要一到两周，毕竟要从海外运过来。二是考虑到价格，建议不要对产品质量有过高的预期。不过和拼多多一样，退款还是挺方便的。

----

一元店
----------------

温哥华有不少一元店，可以理解为线下版的Temu，几乎是你在线下能买到最便宜百货的地方了。学校里就有一家，叫Dollar 'N' Plus，在麦当劳附近。

:: 
    
    Location: 5721 Dalhousie Rd, Vancouver, BC V6T 2H9

----

二手
-----------

如果想少花钱买到质量不错的东西，购买二手是一个不错的选择，而且也更环保。“别丢掉，再使用”。主流的二手在线交易平台是Facebook的Market，但也要注意防诈骗。此外，可以加入一些Facebook的群组，里面是UBC免费送二手的地方，只要你手快，你都不用花钱买家具了。

----

支付宝
------

温哥华有很多超市和餐厅是接受支付宝付款的。他乡遇故“支”。在Richmond的Costco旁的丰泰超市甚至还有家支付宝旗舰店。此外，缴纳学费不受到每年5万刀的外汇限制，也能通过支付宝缴纳。
